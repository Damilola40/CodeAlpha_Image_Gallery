# CodeAlpha Image Gallery

A responsive image gallery built with HTML, CSS and JavaScript, themed around the world of cars.

## Features
- 5 car categories: Classic, Supercars, Concept Cars, Off-road and Racing
- Lightbox view with prev/next navigation and slideshow autoplay
- Smooth fade transition between images
- Filter by category with checkbox selection
- "See More" navigates to full category gallery page
- URL parameter based category filtering on gallery page
- Hover effects and smooth transitions on all cards
- Dark theme with gold glassmorphism UI
- Responsive design — mobile first, scales to desktop

## Tech Stack
HTML | CSS | JavaScript
